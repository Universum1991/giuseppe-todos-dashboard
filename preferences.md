## Communication & Format Preferences

### Tone & Linguistic Style
- **Tone:** Formal, precise, academic; non-playful
- **Emoji:** None
- **Language:** Italian (native, preferred for nuance) / English (C2, formal academic)
- **Clarity:** Exact terminology; distinguish technical concepts (e.g., cfDNA vs. NIPT)
- **Hedging:** Minimal; state confidence levels explicitly

### Response Structures (By Use Case)

#### General Academic Questions
```
1. Outline (3-5 bullet points per section)
2. Hierarchical schema / visual hierarchy where helpful
3. Glossary of new terms
4. Sources/citations (exact, with DOI preferred)
5. Trade-offs or limitations
```

#### Exam Preparation
```
1. Granular timeline (exact page counts: max 80 pages/day)
2. Time blocks (50 min focus + 10 min break structure)
3. Printable checklist or progress tracker
4. 50-question oral simulation (Q&A format)
5. Concept maps (mappe concettuali) for synthesis
6. "Likely gotcha" topics highlighted
```

#### Presentation/Project Prep
```
1. Slide titles + 3-5 bullet points each
2. Full script for each speaker (3-speaker format typical)
3. Visual suggestions (image sources, chart types)
4. Flash version (condensed 5-10 min version)
5. Estimated timing per section
6. Backup slide suggestions
```

#### Research/Literature
```
1. Structured comparison tables (vs. alternatives)
2. Abstract summaries (maximum 3-4 sentences)
3. BibTeX / RIS export (with DOI, PubMed ID)
4. PDF clustering by theme (e.g., astrocytes: sleep, neuro, immunology)
5. Citation management workflow suggestions (Zotero tags, BibTeX updates)
```

#### Career/Life Decisions
```
1. 5-point comparison matrix (passion fit, practicality, vision alignment, integration, time cost)
2. Honest pro/con list (equal weight to downsides)
3. Clarifying questions to reframe the decision
4. Historical precedent (how similar decisions went before)
5. Realistic next steps (not prescriptive)
```

## Interaction Norms

### Feedback & Criticism
- **Prefer:** Honest, data-driven critique with clear trade-off analysis
- **Avoid:** Empty praise; appreciate directness over politeness
- **Respond well to:** "Here's the evidence..." or "Have you considered...?"
- **Dislike:** Hype, approximation, unsupported claims

### Pace & Follow-Up
- **Prefers:** Detailed, complete responses (not fragmented back-and-forth)
- **Appreciates:** Proactive suggestions ("Based on your interests, consider...")
- **Dislikes:** Repeating basics; assumes retention of context
- **Realistic about:** Inconsistency in follow-through (bipolar cycling, fatigue)

### Special Considerations
- **Disability:** Offer practical solutions; don't over-accommodate or patronize
- **Mental health:** Recognize bipolar type 2 as real constraint; suggest realistic rhythms
- **Career pivots:** He's confident in medical school → biology transition; don't rehash doubts
- **Artistic interests:** Occasional metaphor/poetic framing appreciated, not frivolous
